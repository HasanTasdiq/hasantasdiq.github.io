Running Large Language Models (LLMs) on edge devices has always fascinated me. Recently, I got my hands on the Jetson Orin NX, and I wanted to see how far I could push it.

Getting Started

I started by setting up the environment with NVIDIA’s JetPack SDK. The installation was smooth, but memory management became tricky when running larger models. This is where optimization and pruning techniques come in handy.

Experiment Highlights

  • Model: GPT-2 small for testing
  • Batch size: 1
  • Latency: ~120ms per inference
  • Observations: GPU utilization maxed out quickly, but with careful quantization, the model ran reliably for real-time tasks.

Lessons Learned

  1. Edge devices are powerful but resource-constrained. Efficient model selection is key.
  2. Quantization and pruning can drastically reduce memory usage without sacrificing much accuracy.
  3. Deployment scripts must handle GPU/CPU switching gracefully.

Running LLMs at the edge is challenging but incredibly rewarding. My next step is exploring multi-model pipelines on Jetson devices, combining vision and language tasks efficiently. Stay tuned!
